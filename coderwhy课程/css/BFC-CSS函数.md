### 解决浮动的高度塌陷问题

BFC的高度为auto的情况下 子盒子不同情况的高度计算

行内级元素 高度是行高

块级元素 盒子撑起高度

绝对定位元素 高度就被忽略

浮动元素 高度会增加到包裹所有浮动元素

通过给父元素设置overflow:auto形成BFC
